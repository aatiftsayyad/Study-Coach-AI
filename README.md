# Study Coach AI

The AI Study Coach That Remembers, Adapts, and Guides.

## Powered by Microsoft
Uses Microsoft Azure and Microsoft Foundry Services.
The Web APP is also made using Coding Agent.

## Demo Url
https://oldserver.karad.city/apps/aatif/scaweb/

1) Click on Get Started
2) After that click on Start Journey
3) Fill you information
4) Talk to Study Coach AI.

## Overview

Study Coach AI is an AI-powered learning companion that helps students:

- Create personalized study plans
- Track learning progress
- Generate quizzes
- Identify weak topics
- Build study streaks
- Receive adaptive coaching

## Features

- AI Agent powered coaching
- Memory-based personalization
- Quiz generation
- Progress analytics
- Achievement system
- Personalized study roadmap
